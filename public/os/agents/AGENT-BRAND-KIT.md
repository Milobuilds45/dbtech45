## Usage Guidelines

### Web Usage
```html
<img src="/os/agents/anders.png" alt="Anders - Developer" />
```

### Sizing
- Full size: Original (varies, ~1024px)
- Card size: 128px × 128px
- Avatar size: 48px × 48px
- Favicon size: 32px × 32px

### Background
- Icons designed for **black background** (#000000)
- If using on light background, add dark circular backdrop
